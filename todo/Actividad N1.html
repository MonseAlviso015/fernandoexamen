HTML

HTML son las siglas en Hyper Text Markup Lenguage, que puede traducirse como lenguajes de marcas o marcado de hipertexto

El lenguajeHTML se emplea para crear las paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nombre similar

HTML se componen de etiquetas que se escriben entre los simbolos menor y que mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990. cuando Tim Berners-Lee creo la primera pagina web

Version de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991.

En la actualidad la ultima version de HTML es HTML5.
